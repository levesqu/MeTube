Assignment 1

This application is a simple To-Do List application. I had to translate Code from an old Objective c tutorial from app to make the To-Do list part work properly. You add an item to the list by hitting add in the top right corner and type the item in the text field. Then hit Save. A known bug is that if you go to add and item and hit cancel, it will create a blank item. The next time you add an item it will then have a blank line there in between items.
